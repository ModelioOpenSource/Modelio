You are about to remove a module from the project.

This operation has the following consequences:

  • All annotations (stereotypes, tagged values, notes, etc.) brought by this module will be deleted.
  • All the module's commands and functionnalities won't be available anymore.
Warning: this operation can't be undone, meaning even reinstalling the module won't restore the annotations.

N.B. Some read-only elements can use the module's annotations (stereotypes, tagged values, notes, etc.).
The read-only libraries using the module's annotations have to be removed from the project prior to removing the module itself.
To do so, use the 'Configuration\Libraries...' command.
