Le module Java Designer permet de générer le code source Java à partir d'un projet.


La commande de génération de code du module Java Designer

Etapes :

  1. Séléctionner l'élément à partir duquel vous souhaitez générer le code
  2. Cliquer sur le bouton de génération

Une fois le code généré, vous pouvez l'éditer directement dans Modleio grâce à l'éditeur de code source :


La commande d'édition du module Java Designer

Etapes :

  1. Séléctionner l'élément dont vous souhaitez éditer le code source
  2. Cliquer sur le bouton d'édition
