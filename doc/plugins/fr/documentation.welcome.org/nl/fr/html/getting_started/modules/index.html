Gestion des modules Modelio

Certaines fonctionnalités sont fournies par des modules spécifiques (génération de code, de documentation, etc.). Les modules sont des plugins qui peuvent être ajoutés à Modelio pour étendre ses capacités.
Exemple: pour générer le code Java d'un modèle UML, il faut déployer le module Java Designer.

Où trouver des modules Modelio ?
Les modules Modelio peuvent être téléchargés depuis le store Modelio.

Comment rendre un module disponible pour Modelio ?
Les modules téléchargés doivent être stockés dans le catalogue de modules.

Le catalogue de modules Modleio

Etapes :

  1. Lancer la commande "Configuration / Catalogue de modules..."
  2. Dans la boîte Catalogue de modules, cliquer sur "Ajouter un module au catalogue" et utiliser l'explorateur de fichier pour sélectionner les modules (fichiers *.jmdac)
  3. Cliquer sur le bouton "Fermer" lorsque le catalogue est à jour.

Déployer un module dans un projet
Pour déployer un module dans un projet, utilisez la boîte de configuration du projet :

Légende :

  1. Liste des modules disponible pour le projet
  2. Liste des modules du catalogue de modules
  3. Bouton d'extension/réduction de la vue Catalogue
  4. Propriétés du module sélectionné

Etapes :

  1. Ouvrir la vue Catalogue en cliquant sur le bouton
  2. Sélectionner un module dans le catalogue
  3. Cliquer sur le bouton pour ajouter le module à la liste des modules du projet
  4. Les modules du projet peuvent être activés et désactivés en cochant/décochant la case correspondante
  5. Les paramètres des modules peuvent être édités dans la vue Paramètres des modules du projet
  6. Un module peut être retiré de la liste des modules du projet. Pour cela, il faut sélectionner le module et cliquer sur le bouton
