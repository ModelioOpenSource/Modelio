= R2030




===== Résumé

Dans un _PropertyContainer_, le nom de chaque _EnumerationPropertyType_ doit être unique.




===== Détails

Avoir plusieurs éléments portant le même nom peut être déroutant.




===== Conseils

Renommez chaque élément ayant un nom dupliqué.
