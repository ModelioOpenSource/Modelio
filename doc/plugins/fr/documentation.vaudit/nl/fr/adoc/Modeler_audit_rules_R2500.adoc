= R2500




===== Résumé

Un _Parameter_ de type 'out' ne peut pas avoir de valeur par défaut.




===== Détails

Un paramètre de sortie ne peut pas avoir de valeur par défaut.




===== Conseils

* Supprimez la valeur par défaut.
* Changez le type de paramètre en 'in'.
