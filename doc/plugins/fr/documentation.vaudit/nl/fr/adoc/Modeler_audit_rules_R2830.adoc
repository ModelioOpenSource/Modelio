= R2830




===== Résumé

La source d'une transition ne peut être un état final.




===== Détails

Un état final est le dernier état d'un objet, il ne peut pas cibler d'autres états.




===== Conseils

Si vous avez une transition venant d'un état final, supprimez-la.
