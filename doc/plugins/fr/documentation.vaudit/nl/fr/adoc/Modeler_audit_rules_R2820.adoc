= R2820




===== Résumé

La cible d'une transition ne peut pas être un état initial.




===== Détails

Un état initial est un état temporaire de départ et ne peut donc pas être ciblé par une transition.




===== Conseils

Si vous avez une transition ciblant un état initial, supprimez-la.
