= R1530




===== Résumé

Certains éléments devraient avoir un nom.




===== Détails

Les _Associations_ et les _Ports_ devraient avoir un nom.




===== Conseils

Si vous avez une _Association_ ou un _Port_ sans nom, vous devriez lui en donner un.
