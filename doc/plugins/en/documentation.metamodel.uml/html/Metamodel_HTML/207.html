Abstraction
Relationship relating two Elements that represent the same concept at different levels of abstraction.
An Abstraction is a relationship that relates two Elements or sets of Elements that represent the same concept at different levels of abstraction or from different viewpoints. An Abstraction is a Dependency in which the supplier and the client are mapped.
