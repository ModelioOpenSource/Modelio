BpmnDataObject
The primary construct for modeling data within the Process flow is the DataObject element. A DataObject has a well-defined lifecycle, with resulting visibility constraints. Data Object elements must be contained within Process or Sub-Process elements. Data Object elements are visible in a Process diagram.
