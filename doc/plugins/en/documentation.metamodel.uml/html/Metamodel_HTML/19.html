BpmnUserTask
A User Task is a typical "workflow" Task where a human performer performs the Task with the assistance of a software application and is scheduled through a task list manager of some sort. A User Task is executed by and managed by a business process runtime. Attributes concerning the human involvement,like people assignments and UI rendering can be specified in great detail.
