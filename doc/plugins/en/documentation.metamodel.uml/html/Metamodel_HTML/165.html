GeneralOrdering

This mechanism provides the ability to define partial orders of OccurrenceSpecifications that may otherwise not have a specified order.

A GeneralOrdering may appear anywhere in an Interaction, but only between OccurrenceSpecifications.

Figure 147 : GeneralOrdering (architecture_autodiagram)
Association
Description
Before : OccurrenceSpecification [1..1]
After : OccurrenceSpecification [1..1]
The OccurrenceSpecification referred comes after the OccurrenceSpecification referred by before.
