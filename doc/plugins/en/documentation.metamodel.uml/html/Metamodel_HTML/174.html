PartDecomposition
Description of the internal interactions of one Lifeline relative to an Interaction.
A Lifeline has a class associated as the type of the Instance that the Lifeline represents. That class may have an internal structure and the PartDecomposition is an Interaction that describes the behavior of that internal structure relative to the Interaction where the decomposition is referenced. A PartDecomposition is a specialization of InteractionUse. It associates with the Instance that it decomposes.
