The Java Designer module lets you you generate the Java source code of your project.


The Java Designer Generate command

Steps:

  1. Select the element from which you want to generate the code
  2. Click on the Generate button

Once the code has been generated, you can view it within Modelio thanks to the internal code editor:


The Java Designer Edition command

Steps:

  1. Select the element which code you want to edit
  2. Click on the Edit button
