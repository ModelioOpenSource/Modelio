Project configuration

Click on the button to open the Project Configuration window:

Key:

  1. The 'General' section, used to show the Name of the project, and to edit its Description, Images and Contact
  2. The 'Storage' section shows the storage details (project location, size and last modifications)
  3. The 'Models' section shows the different models composing the project
  4. The 'Project's modules shows the modules installed in the project
