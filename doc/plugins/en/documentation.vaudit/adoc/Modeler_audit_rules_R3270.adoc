= R3270




===== Summary

The _State_ of a _BpmnItemAwareElement_ must belong to its represented _GeneralClass_.




===== Details

If a _BpmnItemAwareElement_ represents a _GeneralClass_, then its _State_ must be owned by that _GeneralClass_.
