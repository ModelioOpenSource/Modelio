= R2220




===== Summary

A leaf _NameSpace_ cannot be abstract.




===== Details

A leaf _NameSpace_ cannot be further derived, so it must be a concrete _NameSpace_.




===== Tips

Make the _NameSpace_ non-final or make it concrete.
